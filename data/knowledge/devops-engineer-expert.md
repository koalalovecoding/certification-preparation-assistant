# Microsoft Certified: DevOps Engineer Expert (Synthetic)

## Overview

| Field | Details |
|-------|---------|
| Certification Code | DEVOPS-ENGINEER-EXPERT |
| Certification Name | Microsoft Certified: DevOps Engineer Expert |
| Type | **Certification Bundle** — requires multiple exams/certifications |
| Target Roles | DevOps Engineer, Site Reliability Engineer, Platform Engineer |
| Certification Level | Expert |
| Validity Period | 1 year (renewable via free online assessment on Microsoft Learn) |
| Retirement Status | Active — No scheduled retirement date |
| Renewal Required | Yes — Annual |
| Renewal Method | Free online assessment on Microsoft Learn |

---

## What is a Certification Bundle?

This is a **certification bundle**, not a single exam. It cannot be earned by passing one exam alone. To earn this certification, a candidate must satisfy all requirements listed below.

---

## Requirements

To earn the Microsoft Certified: DevOps Engineer Expert certification, a candidate must:

| Requirement | Details |
|-------------|---------|
| Pass AZ-400 | Designing and Implementing Microsoft DevOps Solutions |
| Hold one of the following active certifications | AZ-104 (Azure Administrator Associate) **or** AZ-204 (Azure Developer Associate) |

### Important Notes
- The AZ-400 exam can be taken before earning AZ-104 or AZ-204, but the Expert certification will not be awarded until all requirements are active.
- Both AZ-104 and AZ-204 satisfy the prerequisite — only one is needed.
- **AZ-204 is retiring on July 31, 2026.** After that date, AZ-104 will be the only valid prerequisite path.
- All certifications must be **active** (not expired) at the time of earning the Expert badge.

---

## Recommended Paths

**Path 1 — Administrator route:**
1. Pass **AZ-104** (Azure Administrator Associate)
2. Pass **AZ-400** (DevOps Solutions)
3. Earn **DevOps Engineer Expert** automatically

**Path 2 — Developer route (before July 31, 2026 only):**
1. Pass **AZ-204** (Azure Developer Associate)
2. Pass **AZ-400** (DevOps Solutions)
3. Earn **DevOps Engineer Expert** automatically
