# Microsoft Certified: Azure Solutions Architect Expert (Synthetic)

## Overview

| Field | Details |
|-------|---------|
| Certification Code | AZURE-SOLUTIONS-ARCHITECT-EXPERT |
| Certification Name | Microsoft Certified: Azure Solutions Architect Expert |
| Type | **Certification Bundle** — requires multiple exams/certifications |
| Target Roles | Solutions Architect, Cloud Architect |
| Certification Level | Expert |
| Validity Period | 1 year (renewable via free online assessment on Microsoft Learn) |
| Retirement Status | Active — No scheduled retirement date |
| Renewal Required | Yes — Annual |
| Renewal Method | Free online assessment on Microsoft Learn |

---

## What is a Certification Bundle?

This is a **certification bundle**, not a single exam. It cannot be earned by passing one exam alone. To earn this certification, a candidate must satisfy all requirements listed below.

---

## Requirements

To earn the Microsoft Certified: Azure Solutions Architect Expert certification, a candidate must hold **all** of the following active certifications:

| Requirement | Certification | How to Earn |
|-------------|--------------|-------------|
| 1 | Microsoft Certified: Azure Administrator Associate | Pass AZ-104 |
| 2 | AZ-305: Designing Microsoft Azure Infrastructure Solutions | Pass AZ-305 |

### Important Notes
- Both certifications must be **active** (not expired) at the time of earning the Expert badge.
- The AZ-305 exam can be taken before AZ-104, but the Expert certification will not be awarded until both are active.
- If AZ-104 expires after earning the Expert certification, the Expert certification will also become invalid until AZ-104 is renewed.

---

## Recommended Path

For most candidates, the recommended order is:

1. Pass **AZ-104** (Azure Administrator Associate)
2. Pass **AZ-305** (Designing Microsoft Azure Infrastructure Solutions)
3. Earn the **Azure Solutions Architect Expert** certification bundle automatically
